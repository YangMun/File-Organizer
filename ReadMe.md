# File Organizer

파일을 종류별로 자동 정리해주는 Ios 앱입니다.

## 📌 주요 기능

- 파일 자동 분류 및 정리
- 파일 종류별 관리 (PDF, 워드, 한글, 엑셀, PPT)
- 최근 업로드된 파일 목록 제공
- 파일 드래그 앤 드롭 지원
- 사용자 설정 기능

## 🛠 개발 환경

- **Language**: Swift
- **Framework**: SwiftUI
- **IDE**: Xcode 16.0+
